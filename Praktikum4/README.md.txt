1. Latihan
	Program membuat list baru dengan nama a = [1, 2, 3, 4, 5]. 
Lalu program memanggil elemen ke 3 yang ada di index no-2, memanggil elemen ke 2-4 yang ada di index no 1-3, dan memanggil elemen terakhir yang mana itu ada di index ke 4. Program juga mengubah elemen ke 4 menjadi 19, dan mengubahnya lagi menjadi 1 dan elemen terakhir juga diubah menjadi 27. Jadi, untuk sekarang list a = [1, 2, 3, 1, 27]. Program membuat list baru Bernama b yang valuenya mengambil dari list a, b = a[0:2]. Lalu menambahkan string "HelloWordl!" ke list b, dan menambahkan 3 value baru lagi. Dan terakhir program menggabungkan antara list a dan list b, dengan perintah gabungan = a + b, hasilnya adalah List gabungan = [1, 2, 3, 1, 27, 1, 2, 'HelloWorld!', 9, 8, 7].

2. Praktikum
	Program dimulai dengan membuat list kosong bernama daftar_nilai untuk menyimpan data yang diinput oleh user nantinya. Lalu program meminta user untuk menginput data seperti nama, nim, nilai tugas, nilai uts, dan nilai uas. Setelah itu, program akan menanyakan user apakah mau menginput data lagi? Jika iya, maka program akan menampilkan list lagi, dan jika tidak, program akan membuat table yang digunakan untuk menampilkan data seluruh siswa. Dan program juga memulai kalkulasi perhitungan nilai akhir seluruh siswa, setelah berhasil menghitung nilai akhir seluruh siswa, program akan mencetak table dan program selesai.
