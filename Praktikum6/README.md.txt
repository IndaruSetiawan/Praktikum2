Praktikum 6

Program dimulai dengan menginisialisai membuat dictionary kosong dan menampilkan menu Utama dalam loop tak terbatas(while True). Dictionary kosong yang digunakan sebagai database sementara untuk menyimpan seluruh data mahasiswa. Program memiliki 5 fungai Utama, yaitu :
1. Fungsi tambah(), untuk menambahkan data mahasiswa baru ke dalam system. Alur kerjanya : program meminta user untuk menginput NIM, Nama, Nilai Tugas, Nilai UTS, Nilai UAS. Lalu program akan menghitung nilai akhir, dan menyimpan data ke dictionary data_mahasiswa dengan nama sebagai key.
2. Fungsi tampilkan(), unutk menampilkan seluruh data mahasiswa dalam bentuk table yang rapi. Alur kerjanya : Mencetak header, menggunakan loop untuk iterasi setiap data di data_mahasiswa, dan menampilkan nomor urut secara otomatis menggunakan variable count.
3. Fungsi hapus(), untuk menghapus data mahasiswa berdasarkan nama. Alur kerjanya : Program meminta user untuk mengiput nama mahasiswa yang ingin dihapus, lalu program akan mengeceknya apakah nama ada di dictionary menggunakan operator in, jika ditemukan program akan menghapus data mahasiswanya, dan jika tidak program akan menampilkan pesan eror.
4. Fungsi ubah(), untuk mengubah nilai mahasiswa yang sudah ada. Alur kerjanya :  Program meminta user untuk mengiput nama mahasiswa yang ingin diubah, lalu program akan mengeceknya apakah nama ada di dictionary, jika ditemukan program meminta user untuk menginput nilai tugas, UTS, dan UAS baru, menghitung nilai akhir baru, dan menyimpan data baru, dan jika data tidak ditemukan program akan menampilkan pesan eror.
5. Fungsi menu(), fungsi utam yang mengatur alur program dan menampilkan menu pilihan. Alur kerjanya : menjalankan infinite loop dengan while True, menampilkan menu dengan  5 pilihan, meminta user untuk memilih, menggunakan conditional statements untuk memanggil fungsi yang sesuai, dan loop akan terus berjalan sampai user memilih menu 5.
